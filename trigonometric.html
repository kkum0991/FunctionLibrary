Background

Trigonometry was studied back in ancient times, but the trigonometric functions were created during the medieval times. The chord function was used much before sine and cosine, back during the first century BC when it was discovered by Hipparchus of Nicaea and Ptolemy of Roman Egypt. The use of sine and cosine functions, however, can be traced back to the Gupta period, between the years 320 and 550 CE. The abbreviations for these functions were first used by a 16th century French Mathematician named Albert Girard. Euler promoted these names, too. Georg Joachim Rheticus was most likely the first person to use these functions with triangles instead of circles. In the year 1682, a German polymath and philosopher named Gottfried Wilhelm Leibniz proved that the sin function is not an algebraic function in terms of x.

In the real world, sound waves are made out of sine functions. Whenever a bullet is fired out of a gun, a sine or cosine function can be used to figure out how much it moves up and down while travelling.

The two main parent trigonometric functions are y = sin(x) and y = cos(x)


How to Graph

Both the sine and the cosine functions will continue the same pattern forever because they follow the unit circle. Knowing the unit circle is be a great help when graphing the function. It takes 2π to go around the whole circle so with comes key points like the following for sine: (0, 0) ([π/2], 1) (π, 0) ([3π/2], -1) and (2π, 0).
Cosine y coordinates follow the same pattern:
0, 1, 0, -1, 0.
Except instead of starting at the origin it starts at y = 1.
For cosine the x values are the same but they are as follows:
1, 0, -1, 0, 1.


How to Solve (Sine)

sin(x) + √2 = -sin(x)
First, add sin(x) to both sides in order to get the sines together.
2sin(x) + √2 = 0
Next, subtract the square root of 2 from both sides in order to isolate the sines and the “x”.
2sin(x) = -√2
Divide both sides by 2 to leave the sin and “x” on it’s own.
sinx = -√2/2
The simplified version of this equation leaves x equal to 5π/4 and 7π/4 in the interval [0, 2π).
x = 5π/4, 7π/4

2(sin(x))2 = 1
Divide both sides by 2 in order to simplify.
(sin(x))2 = 1/2
Square root both sides in order to get rid of the squaring of the two and isolate the sine and the “x”.
sinx = -1/√2, 1/√2
Simplify by multiplying the top and bottom by the square root.
There cannot be a square root for the denominator of a fraction.
Also, because we square rooted a variable, there is a positive and negative version of what sin(x) is equal to.
sinx = -√2/2, √2/2
The values of this are as follows:
x = 5π/4, 7π/4, π/4, 3π/4

2(sin(x))2 - sin(x) - 1 = 0
This equation is very similar to one that’s quadratic, where you just have “x” instead of sin(x).
All you have to do here is factor in order to get rid of the sin2x.
(2sin(x) + 1)(sin(x) - 1) = 0
Next, set both of the binomials in the parentheses into their own equation in order to solve for “x” to create a zero.
We’ll start with 2sin(x) + 1 = 0 first. Subtract 1 from both sides and then divide both sides by 2 in order to isolate the sin(x).
sin(x) = -1/2
On the radian circle, this value is equal to 7π/6 and 11π/6
x = 7π/6, 11π/6
Next, we have to go back and work with the other binomial that was in the parentheses.
Add 1 to both sides in order to isolate the sinx in the same way.
sin(x) = 1
This is equal to π/2.
x = π/2
In the end, all of these values are solutions to the very first equation.
x = 7π/6, 11π/6, π/2

Transformations

Transformations are the same in terms of every other graph except for horizontal and vertical stretches and compressions. Trig graphs have amplitudes and periods. The amplitude is how far the graph go from the origin and it is a in this equation= f(x) = a [sin/cos] b (x - c) + d. In both parent functions the amplitude is 1. The period is how fast the graph moves around the unit circle. To find period it is (2π/b). If b is smaller than one the rotation around the circle(revolution) will slow down but if b is greater than 1 it will speed up. So in a parent function 1 revolution is 2π. When on a graph it is “valley to valley”. If the period is [½] then every revolution is twice as slow or double the x coordinate just like a horizontal compression.


y = -3(sin(4x - 6)) + 72
Factor out 4 to get the “b” value.
y = -3(sin(4(x - 32)) + 72
There is a vertical stretch by a factor of 3 that changes the range from [-1, 1] to [-3, 3].
The entire function is also reflected over the x-axis.
The period of the sine function is changed from 2π to π/2 because of the horizontal compression by a factor of ¼ (horizontal compressions are reciprocals).
There is a translation to the right by 1.5 units (horizontal translations are opposite).
There is a translation upwards by 3.5 units. The new range is [0.5, 6.5].

y = 12(cos(-x - 1)) - 3
Factor out -1 to get the “b” value.
y = 12(cos(-1(x + 1))) - 3
There is a vertical compression by a factor of 12 that changes the range from [-1, 1] to [-0.5, 0.5].
The function is reflected over the y-axis.
There is a translation to the left by 1 unit (horizontal translations are opposite).
There is a translation downwards by 3 units. The new range is [-3.5, -2.5].
