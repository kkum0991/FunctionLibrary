Logarithmic

Background

The person to first discover logarithmic functions was named John Napier. Logs can be used to find the rate of which something happens. So say you went from 100 dollars to 150 in the span of 5 years. The logarithmic function shows a continuous change which could be used to figure out the growth from unknown years. Logs are used to measure things like earthquakes, sound in decibels, and acidity. Career paths include archeology because they use it to find out how old the fossil is and coroners because they use it to find out how old a body is.


How to Graph

Example
To graph without a calculator you should use these key points. (1, 0) (2, 1), (4, 2) (8, 3). The easy way to remember that is that the points are opposite from an exponential since the log is an inverse of an exponential. So for every one y value the x value from the previous coordinate gets squared.

Key Features of y = log2(x):

  • Asymptote at x=0
  • base of 2
  • Domain (0,∞)
  • Range (-∞,∞)
  • negative (0,1)
  • positive (1,∞)
  • increasing (0,∞)
  • end behaviors as x goes toward infinity y foes toward infinity.
  • As x goes toward zero y goes toward negative infinity.
  • X and Y intercepts
