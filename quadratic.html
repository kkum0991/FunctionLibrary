Quadratic

Background

Pythagoras and Euclid created the general procedure to find solutions to the quadratic equation. But have been dated back to being used in the ax2 = c in ancient Egypt, India, and China. Brahmagupta, an Indian mathematician developed the Formula to become ax2 + bx + c form.

People today who may use the function are police officers. They may use the quadratic equation at the scene of a car accident. The equation is used to determine the velocity at which the vehicles were traveling when the collision happened. It can also help the officer figure out which driver is at fault. Architects for the gateway arch in St. Louis, Missouri used the quadratic equation to make the arch. Also, engineers may use the the function to accurately make cables on a bridges.

The parent function is y = x2.


How to Graph

By following these tips you can graph any quadratic function. There are three ways a quadratic equation can look like.

  • 1. y = ax2 + bx + c
  • 2.y = a(x - c)(x - d)
  • 3.y = a(x - h)2 + k

To graph an accurate graph without a calculator if to find the vertex and x-intercepts and whether or not the graph has a maximum or minimum. If a is negative then the function will have a maximum. If a is positive then the graph will have a minimum. But how to find this extrema. If the function is in the format ax2 + bx + c then one equation for finding the x value is -b⁄2a . Then plug this x value into the function to find the y value. If the format is in 3. a(x - h)2 + k then the vertex is the opposite of h (x-value) and k (y-value). If the format is a(x - c)(x - d) then to need to simplify it into the ax2 + bx + c form and follow the same process to finding the vertex. Quadratic functions are also symmetric around the vertex so f(-x) = f(x) so that can help you find more points onto the graph. Then by finding the x intercepts and connecting it to the extrema you can accurately graph a quadratic function.

Key Features of parent function: y = x2

  • Domain: (-∞, ∞)
  • Range: (-∞, ∞)
  • x-int:(0, 0)
  • y-int:(0, 0)
  • Increasing:(0, ∞)
  • Decreasing:(-∞, 0)
  • Positive: Always
  • Negative: Never
  • Vertex: (0, 0)
  • Axis of symmetry: x = 0

Transformations

Graphical Example: In the equation y = 2(x - 3)2 + 2, since 2 is in at a the graph is being vertically stretched by 2. Since x is being subtracted by 3 the graph is shifted to the right 3. Since k is 2 the graph is being shifted up 2.
