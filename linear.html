Linear

Background

Linear functions are those whose graph is a straight line. A linear function has the following form. y = f(x) = ax + b. The slope (rise over run) of the line is represented by a, while x is the variable, and b is the constant. A linear function has one independent variable and one dependent variable. The independent variable is x and the dependent variable is y.

Linear functions date back to times of the ancient Babylonians. Little is known about their origin. The first person to refer to it as a function was Gottfried Wilhelm Leibniz.

Linear functions are useful when determining revenue when fixed costs are involved. They can also be used when calculating distance and/or time. In fact, linear functions have use almost anywhere where constant growth/decay is involved, because of the fact that a linear function increases at the same rate evrerywhere.


How to Graph

The slope, a, is the rise over run. If the a value is 3 for example, for every 1 step to the right, the y value goes up 3. If a was ½, for every 2 you go right, the y value goes up 1. Similarly, if a is negative, like -5, for every 1 we go right, the y value goes down 5.

One important thing to note when determining intercepts are parentheses. Both y = 5(x + 5) and y = 5x + 5 look similar, but the parentheses make y = 5(x + 5) actually y = 5x + 25. The b value is the y-intercept, as well as the x-intercept divided by the negative a value. This is because of the nature of the linear function. In its base form (y = ax + b), for every 1 right you go one up. This means if the y-intercept is (0, -3) (meaning b is -3 and the equation is y = x - 3) then the next value would be (1, -2) then (2, -1) and then (3, 0), the x-intercept, which is also b divided by the negative a value.   • Use the points (-1, -1), (0, 0), and (1, 1) as anchor points when transforming and/or shifting a linear function
  • Do transformations first (a), then shifts (b)

How to Solve

To solve a linear equation, simply put, you must isolate x so that it has no constants or coefficients on it's side of the equation
here are some examples:

x is multiplied by a coefficient of two, y is 5, what is x?
2x = 5
Divide both sides by two to make the coefficient of x one. Because x is isolated we are finished.
x = 2.5

We look at this problem, and see x with a coefficient of 3, a constant of 27, and y as 81
3x + 27 = 81
Subtract 27 from 81
3x = 54
Divide both sides by 3 because that will make x completely isolated from constants and coefficients on its side
x = 18

Let's look at a more complex example.
129 = 2(3(x + 6)) + 19
First, subtract 19
120 = 2(3(x + 6))
Next, divide by 2
60 = 3(x+6)
We can divide by 3 now
20 = x + 6
Subtract 6
x = 14


Transformations

Take into consideration the following linear function:
 y = 4(3(x - 3)) + 4

We see that this function has gone through a:

  • Vertical strech by 4 (Multiply all y values by 4)
  • Horizontal compression by 3 (Divide all x values by 3)
  • Horizontal shift right 3
  • Vertical shift up 4

Vertical and horizontal transformations are indistinguishable on linear functions due to their constant nature. If you divide x (horizontal compress by 4) by 4 or multiply y by 4 (vertical strectch by 4), you would get the same result in a linear function. In this function, this means both the horizontal compression and vertical stretch effectively multiply to 12. Horizontal shifts are multiplied by vertical and horizontal transformations because they are in parentheses with the x value when factored. Vertical tranformations apply to all variables, constants, and coefficients in the equation.
