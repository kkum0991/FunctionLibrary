Square Root

Background

Interestingly, the square root was first seen on the Yale Babylonian Collection YBC 7289 clay tablet, estimated to be created between the years 1800 BC and 1600 BC. On it, it shows the square root of 2 and also states that the square root of 2 divided by 2 is equal to 1 divided by the square root of 2, which is true. There is also a papyrus written in 1650 BC that shows the method used by egyptians to extract square roots using an inverse proportion method. Because the square root itself is a simple operation, it doesn’t make sense to say that someone found it. Around the years between 800 BC and 500 BC, a text in Ancient India was created and showed a method to approximate the the value of the square roots of 2 and 3. The Ancient Greeks knew about square roots, too. They realized that if a number is positive, whole, and not a perfect square, its square root is an irrational number. This means that they cannot be expressed as a fraction. During the 9th century, Arab writers called the equal factors of a number “jadhr”, and those of medieval Europe called it “radix”. During these years, an Indian mathematician named Mahāvīra was the first person to say that you can’t take a square root of a negative number. This is because there isn’t a number that can be multiplied by itself to produce a negative number. If a positive number is squared, the result is obviously positive. If a negative number is squared, the two negatives cancel out and the result is still positive. The symbol for square roots was first invented in by Regiomontanus. It was written as an elaborate R and represented the previously mentioned word “radix”. Over the course of around 100 years, the symbol kept changing until finally, the symbol used today was printed in Christoph Rudolff’s Coss in the year 1525. From then on, it was maintained as that symbol. Today, square roots are used in the Pythagorean theorem. This is where “a” squared plus “b” squared is equal to “c” squared. This is used to find the side length of a right triangle if two other side lengths are known. They are also used in chemistry for finding the molarity of certain chemical equations when attempting to find equilibrium in a reaction. That’s just one of the ways it’s used by scientists, and many others exist. It’s also quite important when working with statistics, considering it’s needed to find variance, or the average of the squared differences of the mean. This is used to figure out out the spread of a set of data.
The parent function is y = √x


How to Graph

On the parent function (y = √x), the first point is at (1,1) because the square root of 1 is 1. To make the “y” go up by one, the “x” has to increase by the previous number, and two more. For example, after (1,1) is on the graph, to find the next point, you add 1 to “y”, and then add 3 to the “x” that was there before, making it (4,2). Next, you add 1 to “y” again, but you add 5 to the previous “x”, making the point (9,3). You can continue to use this strategy in order graph the parent function as far as you need. The “x” never goes negative on the parent function because the square root of a negative number does not exist. The “y” value never goes negative because you cannot get a negative number from the square root of a positive number. You can also graph the parent function with transformations and change the pattern as needed. If there is a vertical stretch of 2, the “y” value increases by 2 instead of 1 in the pattern. The same thing applies for horizontal stretches or compressions, but they affect the x. If there is a translation, the starting point for this pattern is altered according to how the function was shifted. The general form of the square root function is y = a√(b(x-h))+k, where the “a” value dictates vertical growth, the “b” value dictates horizontal growth, and the “h” and “k” value are the starting point of the equation in the form (h,k). Changing the “a” or “b” value results in a vertical or horizontal stretch / compression, and changing the “h” and / or “k” value results in a translation. Whenever you do a vertical stretch or compression, the “k” value is also affected, and whenever you do a horizontal stretch or compression, the “h” value is affected, too. For example, if you have the function y = 2√x + 2, and you wanted to give it a vertical stretch of 2, you would multiply the entirety of the function by 2, leaving the function as y = 4√x + 4. Interestingly, if you want to do a horizontal stretch or compression, you have to input the reciprocal of the stretch or compression you want to do. If you want to do a horizontal compression of 1⁄3, you would input 3 as the “b”. y = √3x+2 would turn into y = 3√(3(x+2)), which simplifies to y = 3√(3x+6). As a side note, whenever the “a” or “b” value is 1, it is not visible in the function as it has no effect (these values are always multiplied). If the “a” value is negative, a reflection over the x-axis is created, and if the “b” value of y=a√(b(x-h))+k is negative, the function is reflected over the y-axis.

4x = 1024
Divide both sides by 4 in order to isolate “x”.
x = 256


Square root functions can be simplified if the number in the parentheses has a factor that is a perfect square. For example, the square root of 27 can be written as 3√3 because 9 is a factor of 27, so it can be square rooted and then pulled to the outside of the square root symbol.

Transformations

y = √-2(4(x - 3)) + 7

  • First, there is a vertical stretch by a factor of 2, alongside a reflection over the x-axis. Every “y” value in the parent function is multiplied by -2.
  • Next, there is a horizontal compression by a factor of 1⁄4 (Horizontal stretches and compressions are reciprocals). Every “x” value in the parent function is multiplied by ¼.
  • After that, there is a translation upwards by 7 units. Each point on the parent function adds 7 to their “y” value.
  • Finally, there is a translation to the right by 3 units (Horizontal translations are opposites). Each point on the parent function adds 3 to their “x” value.
