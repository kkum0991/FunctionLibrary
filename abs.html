Absolute Value

Background

Jean-Robert Argand was the first person to use absolute value however he used the term module. However, Karl Weierstrass was the first person to use the term absolute value. This function can be used to find thing that bank off other things like in billiards. Since the absolute value is the distance from zero all number will be positive. Absolute values can be used in real life such as a geophysicist uses absolute value to look at the total amount of energy used. In an energy wave, there are both negative and positive directions of movement. Another example is when scuba divers discuss their location in regards to sea level. “50 feet below sea level” doesn’t have to be represented as -50 feet. Lastly, when talking about distances in real life, we usually don’t represent distances in negative numbers, even if the distance covered can be represented on a number line or graph. Walking 100 feet to the west can be described as walking +100 feet, and walking 100 feet to the east can be described as walking -100 feet. We don’t represent the distance using positive and negative numbers, but more in direction.


How to Graph

Graph
y = |x|
To graph, the easiest way is to think of the absolute value function as linear because the rate of change will be constant. Then, reflect the output values of -x coordinates over the y axis. Key points include (-1,1) , (0,0), and (1,1).

Key Features on original absolute vlaue function:

  • Vertex at (0,0) [where it splits into two parts]
  • Domain (-∞, ∞)
  • Range (0, ∞)
  • Negative never
  • Positive all real numbers
  • Decreasing (-∞, 0)
  • Increasing (0, ∞)
  • x and y intercepts
  • End behavior = as x goes toward infinity y goes toward infinity; as x goes toward negative infinity y goes toward infinity.
